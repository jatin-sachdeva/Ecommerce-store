Authentication is done using session and cookie based approach:
    the random id generated by us using crypto module method is used as session ID.
    all this is done using cookie-session library used as middleware
    **https://www.npmjs.com/package/cookie-session**


Project Structure
    The structure is divided in three parts:
        --The routes = all the code related to the http requests are stored here
        --The views =  all the html code that has to be sent is stored here
        --The repository = all the database realted code is stored here
